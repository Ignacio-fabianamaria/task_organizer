# task_organizer

O Task Organizer é um aplicativo simples de lista de tarefas que permite aos usuários organizarem suas tarefas de maneira eficiente.

## Funcionalidades

- Adicionar tarefas.
- Marque tarefas como concluídas.
- Edite tarefas existentes.
- Remova tarefas da lista.
- Exibe data e hora corrente.
